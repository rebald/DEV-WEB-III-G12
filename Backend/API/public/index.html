API

Cette page répertorie toutes les commandes que l'on va retrouver dans cette api ainsi que leur fonctionnement.

API GET

/test

Renvoie la table "codepostal".
Données : idCodePostal, nomVille, province.

/utilisateur

Renvoie la table "utilisateurs", le paramètre idUtilisateur ou mailUtilisateur peut être spécifié pour renvoyer l'utilisateur concerné.
Données : idUtilisateur, nomUtilisateur, prenomUtilisateur, mailUtilisateur, mdpUtilisateur.

/categorie

Renvoie la table "categories".
Données : idCategorie, nomCategorie

/image

Renvoie la table "photo", le paramètre idPhoto ou idUtilisateur peut être spécifié pour renvoyer la/les photo(s) concernée(s).
Données : idPhoto, idUtilisateur, image, idCatgorie, description, date.

/images/:idImg

API POST

/inscription

Ajoute un utilisateur à la table "utilisateurs".
Données : nomUtilisateur, prenomUtilisateur, mailUtilisateur, mdpUtilisateur.

/image

Ajoute une image à la table "photo".
Données : idUtilisateur, image, idCategorie.

/categorie.ts

Ajoute une categorie à la table "categories".
Données : nomCategorie.
